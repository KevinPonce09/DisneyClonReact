# Disney Clone App

Este proyecto es un clon de la interfaz de usuario de Disney, desarrollado utilizando React, Vite, Axios y Tailwind CSS. Proporciona una experiencia similar a la de la plataforma de streaming de Disney, mostrando una variedad de películas y series organizadas por género.

## Características

- Visualización de películas y series organizadas por género.
- Carrousel de imágenes destacadas.
- Visualización de contenido de diferentes estudios de producción, como Disney, Pixar, Marvel, Star Wars, etc.
- Funcionalidad de inicio de sesión para acceder al contenido exclusivo.

## Tecnologías Utilizadas

- React: Biblioteca de JavaScript para construir interfaces de usuario.
- Vite: Herramienta de desarrollo rápida para aplicaciones web modernas.
- Tailwind CSS: Framework de CSS utilitario para diseñar interfaces de usuario.
- Axios: Cliente HTTP basado en promesas para realizar solicitudes HTTP en el navegador y en Node.js.
- TMDB API: API de The Movie Database para obtener información sobre películas y series.
